# Telegram Join Cleaner Bot 🤖

This bot automatically deletes "user joined" messages in Telegram groups.

## Features
- Removes join notifications
- Keeps chats clean

## Requirements
- Python 3.x
- python-telegram-bot

## Setup

1. Install dependencies:
   pip install python-telegram-bot

2. Add your bot token in the code

3. Run:
   python bot.py
